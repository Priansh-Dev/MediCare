# MediCare - Doctor Appointment Booking System

A full-stack web application for booking doctor appointments with separate patient and doctor login systems.

## Features

- **Patient Portal**
  - User registration and login
  - Browse available doctors
  - Book appointments
  - View appointment history

- **Doctor Portal**
  - Doctor registration with specialty
  - View scheduled appointments
  - Manage patient records

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Storage**: LocalStorage (ready for backend integration)
- **Design**: Responsive design with modern UI

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Register as a patient or doctor
4. Start booking appointments!
